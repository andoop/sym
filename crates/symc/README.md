# symc

Rust crate: **Sym** compiler / typechecker / interpreter / VM, and the **`sym`** CLI binary.

- **Repository:** <https://github.com/andoop/sym>
- **User-facing docs:** see the [repository README](https://github.com/andoop/sym/blob/main/README.md) and [README.zh.md](https://github.com/andoop/sym/blob/main/README.zh.md) (Chinese).
- **Hacking:** [docs/DEVELOPMENT.md](https://github.com/andoop/sym/blob/main/docs/DEVELOPMENT.md)

```bash
# from workspace root
cargo run -p symc -- run examples/hello.sym
```
