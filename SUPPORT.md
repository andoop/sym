# Support

## Questions & general discussion

- **Bug reports** and **feature requests**: use [GitHub Issues](https://github.com/andoop/sym/issues) with the appropriate template.
- **Security-sensitive issues**: do **not** open a public issue — see [SECURITY.md](SECURITY.md).

There is no guaranteed SLA; the maintainer(s) respond as time allows.

## What to include

For bugs: minimal `.sym`, exact `sym` / `cargo run` command line, OS, `rustc -V`, and expected vs actual behavior.

# 支持与反馈（中文）

- **功能建议、缺陷**：请用 [GitHub Issues](https://github.com/andoop/sym/issues) 并选用模板。
- **安全问题**：勿发公开 issue，请按 [SECURITY.md](SECURITY.md) 私密报告。

维护者尽力回复，但不承诺响应时限。报告缺陷时请附最小复现、命令行、系统与 `rustc -V`。
